## Project: DataScope — Text Classification & Evaluation Platform

### Overview
DataScope is an interactive ML evaluation platform that takes any text 
classification dataset and runs it through multiple approaches (classical 
ML, LLM-based), provides comprehensive evaluation with data-centric 
insights, and surfaces actionable recommendations for improving data 
quality before model tuning.

The goal is to demonstrate end-to-end applied ML thinking: problem 
framing, data exploration, pipeline construction, evaluation design, 
LLM integration, and data-centric AI principles.

This is a portfolio project. It should look polished, be well-documented, 
and demonstrate deep understanding of ML evaluation — not just sklearn 
boilerplate.

---

### Tech Stack
- **Python 3.11+**
- **Streamlit** — frontend UI
- **Scikit-learn** — classical ML pipelines (TF-IDF, LogisticRegression, 
  evaluation metrics)
- **Pandas / NumPy** — data manipulation
- **Plotly** — interactive visualizations (NOT matplotlib)
- **OpenAI API** (gpt-4o-mini) — LLM classification and extraction
- **python-dotenv** — environment variable management

All dependencies should be in a `requirements.txt` file.
